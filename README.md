# ⚡ Winchu · Nexus

**Shape your presence. Manifest your energy.**

A social wellness app that combines aura-based self-improvement, real-time chat, social feed, diary, routines, and more — all powered by Firebase Realtime Database.

---

## ✨ Features

- **Aura System** — Choose up to 3 energies (Focus, Creativity, Vitality, etc.) with custom daily tasks
- **Dashboard** — Track your strength score, streak, and task progress
- **Social Feed** — Post updates with images/videos, like posts, view stories
- **Real-time Chat** — Public chat with online status
- **Diary** — Private journal entries with mood tracking
- **Routines** — Create and save daily routines
- **Profile** — Custom bio and avatar
- **Wallpapers** — Browse and set beautiful Unsplash backgrounds

---

## 🛠️ Tech Stack

- **Frontend**: Vanilla HTML, CSS, JavaScript
- **Backend**: Firebase Realtime Database
- **Hosting**: GitHub Pages / Firebase Hosting ready
